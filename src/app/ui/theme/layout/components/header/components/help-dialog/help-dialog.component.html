О приложении BARINB

Данное приложение представляет собой, демо проект, для демонстрации возможностей Angular вместе с лучшими практиками.

Приложение реализует часть реального приложения Airbnb, где пользователь может просмотреть список доступного жилья на карте или в списке.

Так как это web-приложение, то оно ориентировано на различные платформы. Поэтому мобильная версия отличается от декстопной, методом изменения работы. В мобильной версии можно переключать вид с поиска на карте, на простой список вариантов. В декстопной версии, доступны сразу два варианта.

При поиске недвижимости на карте, при клике на доступный вариант, показывается превью и при клике на картинку, будет открываться полное описание варианта.

Также для демонстрации работы с формами и изменением данных, добавлен раздел с администрированием.

В разделе администрирования, можно создать и отредактировать три базовые сущности: Person (Владелец недвижимости), Building (отель, комплекс) и Room (комната, апартаменты). Каждый собственник владеет зданием. В здании представлены несколько вариантов (комнат), которые может снять/арендовать клиент.
